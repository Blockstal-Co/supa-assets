### Single Icon
```html

       alt="Facebook" 
       width="32" 
       height="32"
       style="display: inline-block;">
</a>
```

## Tips for Email Use

1. **Recommended sizes**: 24x24, 32x32, or 40x40 pixels
2. **Always include alt text** for accessibility
3. **Test across email clients** - Some may require PNG fallbacks
4. **Keep spacing consistent** between icons
5. **Link to your actual profiles** - Update the href URLs

## Brand Guidelines

These icons use official brand colors. Always check individual platform brand guidelines before use in commercial applications.
